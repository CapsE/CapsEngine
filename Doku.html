$text:

Erläuterung

Setzt den Text des Raums auf den angegebenen Text.

Beispiel

$text: Dies ist ein Text der im Spielfenster zentral angezeigt wird.

$background:

Erläuterung

Ändert das Hintergrundbild zu dem Bild an dem angegebenen Pfad. Der Pfad ist dabei relativ zur Gamedatei.

Beispiel

$background: Images/Hintergründe/Kirche.jpg

$option:

Erläuterung

Erstellt eine Option die den Spieler bei einem Klick zu dem angegebenen Raum weiterleitet.

Beispiel

$option: Dem Weg folgen => \Kreuzung.txt

Hierbei wird ein Button mit der Beschriftung "Dem Weg folgen". Sobald der Button geklickt wird, wird der Raum "Kreuzung.txt" geladen.

$music:

Erläuterung

Spielt die im Pfad angegebene .wav Datei ab (mp3 wird nicht unterstüzt!). Die Musik wird in einer Endlosschleife gespielt bis eine neue Datei zugewiesen wird. Dies gillt auch über verschiedene Räume hinweg. Soll die Datei nur einmal abgespielt werden sollte $sound verwendet werden.

Beispiel

$music: \ambience.wav

$sound:

Erläuterung

Spielt die im Pfad angegebene .wav Datei ab (mp3 wird nicht unterstüzt!). Die Musik wird einmal komplett abgespielt oder abgebrochen und ersetzt falls ein neuer Raum geladen wird in dem ein neuer Sound zugewiesen wird.

Beispiel

$sound: \footsteps.wav

$dialog:

Erläuterung

Schaltet die GUI in den Dialogmodus um und liest den angegebenen Raum. Der Button um den Dialog zustarten trägt die angegebene Beschriftung

Beispiel

$dialog: Guten Tag! => \Dialogs\Dialog1

$endDialog:

Erläuterung

Schlatet über einen Button die GUI wieder in den normalen Modus. Der Button trägt die angegebene Beschriftung. Dabei wird der Raum angezeigt von dem aus der Dialog begonnen wurde.

Beispiel

$endDialog: Auf Wiedersehen!

$fight:

Erläuterung

Erzeugt einen Button "Kampf beginnen" der wenn er geklickt wird das Kampfsystem startet. Es wird der Angegebene Gegner geladen. Zusätzlich zum Gegner muss noch ein Raum angegeben werden der bei erfolgreichem bestehen des Kampfes geladen wird und einer sollte der Held besiegt werden.

Beispiel

$fight: \Gegner\Wache.txt => \WinRoom => \LooseRoom

$set:

Erläuterung

Setzt das angegebene Attribut auf den angegebenen Wert. Exestiert das Atribut noch nicht wird es erstellt.

Vordefinierte Werte sind:

  • Leben -> Die aktuellen Lebenspunkte des Charakters
  • Speed -> Die Geschwindigkeit des Charakters, die über den ersten Angriff in einem Kampf entscheidet

    • Beispiel

    $set: Leben => 3
